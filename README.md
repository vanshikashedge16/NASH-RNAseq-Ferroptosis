# NASH-RNAseq-Ferroptosis
# NASH RNA-seq Analysis: Ferroptosis Gene Identification

## 📌 Project Overview

This project analyzes bulk RNA-seq data to identify ferroptosis-related genes involved in Non-Alcoholic Steatohepatitis (NASH).

## 🧬 Objectives

* Perform differential gene expression analysis using DESeq2
* Identify ferroptosis-related genes
* Visualize results using a volcano plot

## 🛠️ Tools & Technologies

* R Programming
* DESeq2
* ggplot2
* AnnotationDbi

## 📊 Results

Key ferroptosis-related genes identified:

* GPX4
* SLC7A11
* ACSL4
* GCLC
* NCOA4

## 📁 Files Included

* `analysis.R` → complete analysis workflow
* `ferroptosis_NASH_results.csv` → processed results
* `volcano_plot.png` → visualization

## 📌 Conclusion

The analysis highlights the role of ferroptosis in NASH, with significant changes in genes involved in oxidative stress and iron metabolism.
